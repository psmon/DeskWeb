# 개발지침
- 게임종료조건 적용
  - 현재 장군이 포위 당해 움직일 공간이 없는경우 게임이 종료되지않고, AI에서 계속 고민을 합니다.
    - 게임종료 체크 로직을 추가하고 누가 이겼는지? 승/패를 알리는 메시지를 추가해주세요
      - 게임렌더링 내에서 승패를 알려주고, 스코어를 표시하도록 해주세요 
    - 게임종료후에는 새게임을 다시 시작하시겠습니까? 메시지와 함께 이미 구현된 New Game을 수행해주세요
- 창모드일때 
  - 현재 가로/세로 리사이즈를 다음과 같이개선
    - 게임렌더링을 꽉찬 화면으로 해주세요
    - 비율을 고려하지 않고 꽉 채우면 됩니다.
- 추가로 전체모드도 추가해주세요 
  - 전체모드일때는 장기판의 비율을 고려 가장 크기가 큰 화면으로 렌더링 해주세요
  - ESC키를 누르면 창모드로 복귀하도록 해주세요
- 전반적인 UI개선 
  - 현재 세로길이가 상대적으로 길어서.. 와이드 모니터에 잘림현상이 생깁니다.
    - 전체적인 UI를 재배치해 세로길이를 줄여주세요
    - 배치구성시 와이드 비율을 고려해 좌우여백을 적절히 활용해주세요
    - 좌쪽패널 게임, 우측패널 정보성, 하단패널은 AI채팅창을 배치하는 레이아웃을 고려해주세요
  - 게임중에도 카메라앵글을 쉽게 조정할수 있게 게임내 카메라 조정 심플버튼을 게임렌더링 우측상단에 추가해주세요 
    - 게임화면중 html로 구현된 카메라제어 요소는 제거합니다.
    - 설정에도 별도 카메라 앵글설정이 있으며 이 부분은 유지합니다.

## 개선
- AI대화가 개행이안되고 우측으로 계속 붙습니다.
  - 개행이되게하고 3줄까지 표현하고 스크롤 바가 생기도록 개선해주세요
  - 스크롤바는 대화가 추가되면 항상 맨아래로 스크롤되도록 개선해주세요
- 채팅시 LLM대기중일때 "음 생각중입니다" 채팅이 나옵니다. 이 채팅은 없애주고 다음과같이개선
  - 게임뷰 왼쪽에 대국턴을 보여주는것을 추가해주세요
  - AI턴일때는 "AI생각중..." 이라는 메시지가 게임뷰 왼쪽에 표시되도록 해주세요
    - AI생각중은 로딩바느낌으로 애니메이션 효과도 추가해주세요 
  - 사람턴일때는 "당신의 턴입니다." 라는 메시지가 게임뷰 왼쪽에 표시되도록 해주세요
- 턴표시영역 게임이 시작되기전에는 "새게임을 시작해주세요로" , 게임이 시작도 안되었는데 당신의 턴입니다가 표시됨
- 카메라 뷰제어는 마우스로도 가능하게 - 게임뷰 위에서 작동
  - 스크롤 휠로 줌인/줌아웃
  - 마우스 좌클릭 드래그로 앵글조정 ( 좌우,위아래 마우스 움직임 델타값 에따라 카메라 각도조정)
    - 장기말 선택판정 안될때
