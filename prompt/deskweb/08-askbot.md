이 프로젝트는 윈도우OS 데스크탑 컨셉을 경량화된 웹OS UX 어플리케이션으로 구현한것으로
사용자는 브라우저에서 데스크탑 환경을 경험할 수 있는 목표로 제작되고 있으며

기존 프로젝트를 분석을 먼저진행후,  추가기능을 구현해주세요

# 추가기능
- ASK BOT이라는 앱을 추가로 만들고자합니다.
  - iframe기반으로 외부 URL을 로드하는 기능을 구현해주세요
  - https://mcp.webnori.com/ui/askbot 이 URL을 로드하는 기능을 구현해주세요
  - ASK BOT 어플리케이션은 창으로 열려야 합니다. 리사이즈를 지원합니다.
  - 최초 창 사이즈는 모바일 화면 비율 크기로 열리도록 구현해주세요
    - 가로 400px , 세로 700px 
    - 리사이즈를 하면 마지막 크기를 기억하도록 구현해주세요
- 바탕화면에서 우클릭 앱추가를 하면 앱이름과 url을 입력하면 해당 URL을 로드하는 앱이 추가되도록 구현해주세요
  - 앱이름 : ASK BOT 
  - URL : https://mcp.webnori.com/ui/askbot
  - 아이콘은 추가할시 favicon.ico 를 불러오도록 구현해주세요
- ASKBOT은 기본설치 앱이며.. 이 앱을 삭제할수는 없습니다.
- 사용자 추가한 앱의 경우 삭제가 가능합니다. 우클릭 메뉴에 삭제기능을 추가해주세요

# 로컬 테스트
- 구현이 다되며 docker-compose.yml , deskweb-dev 모드로 실행해주세요
  - 실행및 빌드시 발생하는 오류를 모두 해결해주세요
  - 실행이되면 브라우저에서 접속및 정상동작하는지 확인해주세요
  - 실행 확인을 위해 개발모드에서 작동이 잘되는지 확인을 위한 로그를 개발시 꼼꼼히 작성해주세요
  - 코드수정후 반영시 도커재시작없이 docker exec deskweb-dev qx compile 명령을 활용해주세요 - dev모드
- 기본 로컬테스트가 수행되면 직접확인후 피드백 예정으로 추가 수정이 있을수 있습니다. 작동중인 어플리케이션을 유지해주세요
