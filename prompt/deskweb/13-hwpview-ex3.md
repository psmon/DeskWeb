
이 프로젝트는 윈도우OS 데스크탑 컨셉을 경량화된 웹OS UX 어플리케이션으로 구현한것으로
사용자는 브라우저에서 데스크탑 환경을 경험할 수 있는 목표로 제작되고 있으며

기존 프로젝트를 분석을 먼저진행해 추가기능을 구현해주세요
hwpview가 초기버전 작성되어 있으며 개선을 하려합니다.

# 추가기능
- 표기능의 셀병합이의 순서가 제대로 안된것같습니다. 표그리기를 먼저 개선합니다.
```
첫번째표
1. 인적 사항
+------------+-------------+
|            |   성명      |
|            +-------------+
|            |   휴대전화   |
+------------+-------------+

팁 : 3x5 셀구성으로 첫번째표의 첫열은 병합되었습니다. 

두번째표
2. 학력 사항
+--------+--------+--------+--------+--------+
|  재학 기간  | 학교명  |  전공학과  |  평점  |  상태  |
+--------+--------+--------+--------+--------+
|      |      |    |    |    |
+--------+--------+--------+--------+--------+
|    |    |   |  |      |
+--------+--------+--------+--------+--------+
팁 : 3x5 셀로  병합없이 모두 개별셀입니다.

```
- 셀그리기에 성공하면 다음을 개선해주세요
  - 2.학력 사항이 첫번째 표의 셀에들어가거나 사라집니다.
  - 학려사항이 사라지면,재학 기간,학교명 등의 문구도 함께 사라집니다.
  - 사라지지 않으면 문자들이 다른위치의 셀에 들어가게됩니다.
- 한방에 못고칠수 있으니 문제추적을 위한 로그(웹 콘솔로그) 꼼꼼히 작성해주세요
  - 해당로그는 직접실행해 피드백용도로 이용될예정입니다.
