이 프로젝트는 윈도우OS 데스크탑 컨셉을 경량화된 웹OS UX 어플리케이션으로 구현한것으로
사용자는 브라우저에서 데스크탑 환경을 경험할 수 있는 목표로 제작되고 있으며

# 개발지침
- 마우스 우클릭하면 Add App 기능이 있습니다.
- Appname과 URL을 입력하면 바탕화면내 웹으로 연결되는 아이콘이생깁니다.
  - 기본아이콘이 생기며.. favicon 아이콘이 표시될수 있도록 개선해주세요 
- 아이콘 선택 개선
  - 현재 아이콘을 좌클릭으로 선택하면 선택이된지 안된지 알수 없습니다. 배경에 투명한 네모로 선택오버레이를 주세요
  - 우 클릭을하면 delete app이
  - Custome APP인경우 우클릭하면 아이콘 변경기능을 추가해 아이콘을 변경가능하게 해주세요
    - 몇가지 선택가능한 아이콘 시리즈를 만든후 선택하게 해주세요
    - favicon 은 첫번째로 선택가능한 옵션입니다.

# 로컬 테스트
- 구현이 다되며 docker-compose.yml , deskweb-dev 모드로 실행해주세요
    - 빌드오류만및 도커실행만 잘해주세요
    - 코드수정후 재반영시에는 도커재시작없이 docker exec deskweb-dev qx compile 명령을 활용해주세요 - dev모드로 적용됩니다.
- 실행이된후 테스트는 직접검수예정으로 피드백 대기합니다.
