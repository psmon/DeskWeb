프로젝트 설명
- 순수 .js로 작동하며 웹어셈블리를 활용하여 hwp 파일을 해독하고 html로 렌더링하는 모듈입니다.
- file 선택을 하면 hwp 파일을 읽고, 웹어셈블리로 해독한 후, html로 변환하여 표시합니다.
- hwp-wasm-viewer 모듈은 다음과 같은 주요 기능을 포함합니다:
  - .hwp 파일 읽기: 사용자가 선택한 .hwp 파일을 읽어들입니다.
  - 웹어셈블리 해독: 읽어들인 .hwp 파일을 웹어셈블리를 활용하여 해독합니다.
  - HTML 변환: 해독된 데이터를 HTML 형식으로 변환합니다.
  - 렌더링: 변환된 HTML을 웹 페이지에 표시합니다.
