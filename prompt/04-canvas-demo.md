이 프로젝트는 윈도우OS 데스크탑 컨셉을 경량화된 웹OS UX 어플리케이션으로 구현한것으로
사용자는 브라우저에서 데스크탑 환경을 경험할 수 있는 목표로 제작되고 있으며

기존 프로젝트를 분석을 먼저진행후,  추가기능을 구현해주세요

## 추가기능
- Canvas 데모 어플리케이션을 추가하려고 합니다.
  - Canvas 데모 어플리케이션은 HTML5 Canvas를 사용하여 그래픽을 그리는 기능을 제공해야 합니다.
  - Canvas 데모 어플리케이션은 윈도우 화면보호기에 사용된 다양한 수학적 패턴과 애니메이션을 포함해야 합니다.
  - Canvas 데모 어플리케이션은 창으로 열려야 합니다.
  - Canvas 데모 어플리케이션은 사용자 인터페이스가 직관적이고 사용하기 쉬워야 합니다.
  - Canvas 데모 어플리케이션은 반응형 디자인을 적용하여 다양한 화면 크기에서 잘 작동해야 합니다.
  - Canvas 데모 어플리케이션은 다음과 같은 패턴과 애니메이션을 포함해야 합니다: 
    - 원형 패턴
    - 나선형 패턴
    - 파동 패턴
    - 프랙탈 패턴
    - 기타 수학적 패턴
  - 각 패턴은 사용자가 선택할 수 있어야 하며, 패턴의 속도, 색상, 크기 등을 조절할 수 있는 옵션을 제공해야 합니다.
  - 패턴은 부드럽고 연속적인 애니메이션으로 표시되어야 합니다.
  - 패턴 전환 시 부드러운 페이드 인/아웃 효과를 적용해야 합니다.
  - GPU 가속을 활용하여 성능을 최적화해야 합니다. GPU가속이 불가능한경우 CPU모드로 fallback 되도록 구현해주세요.
  - Canvas 데모 어플리케이션은 사용자가 패턴을 전체 화면 모드로 전환할 수 있는 기능을 제공해야 합니다.
