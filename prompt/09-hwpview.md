이 프로젝트는 윈도우OS 데스크탑 컨셉을 경량화된 웹OS UX 어플리케이션으로 구현한것으로
사용자는 브라우저에서 데스크탑 환경을 경험할 수 있는 목표로 제작되고 있으며

기존 프로젝트를 분석을 먼저진행후,  추가기능을 구현해주세요

# 추가기능
- HWP 뷰어 어플리케이션을 추가하려고 합니다.
  - HWP 뷰어 어플리케이션은 .hwp 파일을 열고 내용을 표시하는 기능을 제공해야 합니다.
  - HWP 뷰어 어플리케이션은 창으로 열려야 합니다.
  - HWP 뷰어 어플리케이션은 사용자 인터페이스가 직관적이고 사용하기 쉬워야 합니다.
  - HWP 뷰어 어플리케이션은 반응형 디자인을 적용하여 다양한 화면 크기에서 잘 작동해야 합니다.
  - HWP 뷰어 어플리케이션은 다음과 같은 기능을 포함해야 합니다:
    - .hwp 파일 열기: 사용자가 파일 선택 대화상자를 통해 .hwp 파일을 선택하고 열 수 있어야 합니다.
    - 내용 표시: .hwp 파일의 텍스트와 기본 서식을 올바르게 렌더링하여 표시해야 합니다.
    - 페이지 탐색: 여러 페이지로 구성된 .hwp 문서의 경우 페이지 간 이동 기능을 제공해야 합니다.
    - 확대/축소: 사용자가 문서 내용을 확대하거나 축소할 수 있는 기능을 제공해야 합니다.
    - 인쇄 기능: 사용자가 문서를 인쇄할 수 있는 기능을 제공해야 합니다.
  - HWP 뷰어 어플리케이션은 오픈 소스 라이브러리나 기존 도구를 활용하여 .hwp 파일 형식을 처리할 수 있습니다.
  - 실제 나의 컴퓨터에서 hwp파일을 드래그앤 드랍을 하면 , DESK WEB에 업로드가 됩니다. 반대로 다운로드도 지원합니다.
- HWP 뷰어를 구현하기위한 hwp.js 라이브러리를 활용할수 있습니다.
  - 소스위치 : https://github.com/hahnlee/hwp.js
